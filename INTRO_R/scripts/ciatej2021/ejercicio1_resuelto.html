Ejercicio 1.1

Econtrar los genes signigicativos UP and DOWN a diferentes thresholds de significancia (FDR 0.1, 0.05, 0.001, FC > 2). Hacer gráfico de barras comparativo.

El número de genes regulados significativamente va a depender de los thresholds que establezcamos. Por ejemplo, un threshold de FDR=0.001 es más estringente que un threshold de FDR=0.1, por lo que menos genes serían catalogados como significativos en el primer caso.

Vamos a etiquetar los genes como:

  • “Up-regulated”, o regulados hacia, si tienen un fold change (FC) mayor o igual que 2 y un FDR menor a un threshold determinado.
  • “Down-regulated”, o regulados hacia abajo, si tienen un FC menor o igual que -2 y un FDR menor a un threshold determinado.
  • “Unchanged”, o sin cambios, si no tienen relevancia significativa.

Usaremos tres valores de threshold: 0.001, 0.05, y 0.1.

Definamos una función para ayudarnos a etiquetar los genes:

label_sig <- function(fdr_vctr, fc_vctr, fdr, fc = 2) {
  dplyr::case_when(
    fdr_vctr <= fdr & fc_vctr >= log(fc) ~ "Up-regulated",
    fdr_vctr <= fdr & fc_vctr <= -log(fc) ~ "Down-regulated",
    TRUE ~ "Unchanged"
  )
}

Utilicemos mutate para crear nuevas columnas con las etiquetas. Luego las cambiamos a formato tidy con pivot_longer:

fc_data <- fc_data %>% 
  mutate(
    sig_001 = label_sig(FDR, logFC, 0.001),
    sig_05 = label_sig(FDR, logFC, 0.05),
    sig_1 = label_sig(FDR, logFC, 0.1)
  ) %>% 
      pivot_longer(
        names_to = "sig_threshold", values_to = "sig_label",
        cols = sig_001:sig_1
    )

Ejercicio 2.1

Hacer un volcano plot (comparación FC vs Pvalue) y anotar en el volcano plot los top 10 hacia arriba y hacia abajo. Elegir un threshold de significancia. Anotar con geom_text o geom_label.

Vamos a filtrar únicamente los genes etiquetados con significancia a un threshold de FDR = 0.001:

fc_001 <- fc_data %>% 
  filter(sig_threshold == "sig_001")

Definamos como los top 10 a los genes que hayan sido etiquetados como “Up-” o “Down-regulated” y que tengan el mayor valor de fold change (o menor para el caso de los genes regulados hacia abajo).

fc_001_top10_up <- fc_001 %>% 
  filter(sig_label == "Up-regulated") %>% 
  arrange(-logFC) %>% 
  head(10)

fc_001_top10_down <- fc_001 %>% 
  filter(sig_label == "Down-regulated") %>% 
  arrange(logFC) %>% 
  head(10)

## Join in a single tibble
top10 <- bind_rows(fc_001_top10_up, fc_001_top10_down)

Un volcano plot es un gráfico de dispersión con el logFC en el eje x y el -log10 del P-value en el eje y.

En el gráfico de abajo graficamos primero el dataframe “fc_001” y le agregamos otra capa de geom con los datos en “top10” (la línea con geom_text_repel del paquete ggrepel).

fc_001 %>% 
  ggplot(aes(logFC, -log10(PValue))) +
  geom_point(aes(color = sig_label)) +
  geom_text_repel(
    data = top10, mapping = aes(logFC, -log10(PValue), label = Genes)
  ) +
  theme_bw()

En la línea de código de abajo creamos un tibble en formato tidy:

top10_tidy <- top10_new %>% 
  select(Cpapaya_ID, sig_label, starts_with("L-")) %>% 
  pivot_longer(
    names_to = "Replicate", values_to = "TPM", cols = `L-CNa`:last_col()
  ) %>% 
  arrange(sig_label) %>% 
  mutate(
    Sample = 
      factor(sub("a|b$", "", Replicate), levels = c("L-CN", "L-10", "L-20")),
    # sig_label = factor(sig_label, levels = c("Up-regulated", "Down-regulated")),
    Cpapaya_ID = fct_inorder(Cpapaya_ID)
  ) %>% 
  select(sig_label, Cpapaya_ID, Sample, TPM, -Replicate)
head(top10_tidy)

Ahora creamos un resumen con summarise (media, desviacíon estándar y error estándar):

top10_summary <- top10_tidy %>% 
  group_by(sig_label, Cpapaya_ID, Sample) %>% 
  summarise(
    mean = mean(TPM),
    sd = sd(TPM),
    se = sd(TPM)/sqrt(length(TPM))
  ) %>% 
  ungroup()
